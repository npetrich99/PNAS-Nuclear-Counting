# Nuclear Mask Quantification (Post-Segmentation Analysis)

## Overview:

This script performs automated quantification of nuclei from labeled segmentation masks generated by StarDist2D or other segmentation pipelines.

It reads TIFF segmentation masks, counts unique labeled objects, and outputs a summary Excel report.

This tool is intended as a downstream analysis step following nuclear segmentation.

* Python 3.11+
* NumPy
* pandas

* scikit-image (optional dependency, not required in current version)

Install dependencies:

```bash
pip install numpy pandas tifffile
```

---

## Input Data:

The script expects a directory containing labeled segmentation masks in `.tif` or `.tiff` format.

Important Assumption: Each nucleus must be represented as a unique integer label in the image (e.g., StarDist output or other instance segmentation format).

* Background pixels should be 0
* Nuclei should be labeled 1, 2, 3, ...

Binary masks (0/1) are not appropriate for this script. Input directory is set inside `counting.py` and should be modified by the user if needed.

---

## Usage:

Update the input folder path inside the script:

`folder_path = r"PATH_TO_YOUR_MASK_DIRECTORY"`

Then run:

`python counting.py`

---

## Output:

The script generates:

`Nuclear_Count_Report.xlsx`

This file contains columns with Filenames and Nuclear Mask Count. Each row corresponds to one segmentation mask file. Users should modify the output path if running on a different system.

---

## Method Summary:

For each image:
1. Load TIFF segmentation mask.
2. Extract unique pixel values.
3. Remove background label (0).
4. Count remaining unique labels.
5. Export results to Excel.
